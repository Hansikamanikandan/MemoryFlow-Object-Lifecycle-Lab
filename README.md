# 🧠 MemoryFlow Object Lifecycle Lab

> Designing software is not just about writing code — it is about managing **objects, relationships, and system behavior over time**.

---

## 🚀 Overview

**MemoryFlow Object Lifecycle Lab** is an advanced Python repository focused on exploring how real software systems manage object creation, interaction, memory usage, and lifecycle control.

This project moves beyond academic exercises and models concepts commonly seen in enterprise-scale applications such as backend services, distributed systems, and large production environments.

The goal is simple:

✔ Write clean logic
✔ Model real-world systems
✔ Understand how software behaves internally

## 🧩 System Modules

### 🏫 University Registry Engine

Simulates academic record management while demonstrating nested classes, reference tracking, and controlled object deletion.

### 👩‍💼 Employee Address Composition

Shows real-world **HAS-A relationships**, modelling how enterprise systems attach dependent data objects dynamically.

### 🎓 Course Registration Manager

Implements modular enrollment systems with runtime object allocation and reference inspection.

### 📦 Product Supplier Architecture

Models supplier-product dependency structures used in inventory and ERP platforms.

### 🏢 Corporate Employee Lifecycle

Demonstrates employee onboarding, lifecycle tracking, and memory cleanup operations.
